## HUD

The helicopter HUD is displayed for both the **pilot and co-pilot** whenever they are seated in a cockpit seat. It shows:

| Element | Description |
|---------|-------------|
| **Airspeed** | Current speed in knots |
| **Altitude AGL** | Height above ground level in metres |
| **Health Bar** | Current helicopter HP |
| **Auto Hover Status** | Whether Auto Hover is currently active |
| **Light Status Icons** | Whether the spotlight and navigation lights are on or off |

## Fuel System

### Fuel Capacity & Burn Rate

| Helicopter | Tank Capacity | Burn Rate | Approx. Flight Time |
|------------|--------------|-----------|-------------------|
| MD500 Civilian | 242 L | 126 L/hr | ~1h 55m |
| MH-6 Little Bird | 235 L | 112 L/hr | ~2h 06m |
| SA342M Gazelle | 448 L | 128 L/hr | ~3h 30m |
| UH-1H Iroquois | 833 L | 333 L/hr | ~2h 30m |
| MH-60L Blackhawk | 1,362 L | 545 L/hr | ~2h 30m |
| AW101 Merlin | 3,222 L | 644 L/hr | ~5h 00m |

### Unlimited Fuel

Set `CZH_fuelBurnLPH = 0` to disable fuel consumption entirely.

### Custom Flight Duration Formula

```
CZH_fuelBurnLPH = (CZH_maxFuel / desired_flight_minutes) * 60
```

**Example** — UH-1H for exactly 2 hours of flight:
```
CZH_fuelBurnLPH = (833 / 120) * 60 = 416.5
```

### How to Refuel

Hold a fuel container, approach within `CZH_refuelDistance` metres of the helicopter, and look at the **fuel refill point** on the helicopter's model. The refuel action will appear — activate it to transfer fuel from the container into the tank.

**Accepted fuel sources:**

| Source | Notes |
|--------|-------|
| `CanisterGasoline` (jerry can) | Standard refuel. Default: fills 10% of any tank per full can. |
| `CZH_HeliDrum` (Helicopter Fuel Drum) | Large drum. Default: fills 50% per drum — 2 drums to fill any helicopter from empty. |
| `Bottle_Base` items | Water bottles, glass bottles, etc. Very small fill per bottle. |
| Fuel pump mods | `Simple_Refueling` and `Gas-Pump-Refueling` are both supported — land near a pump and refuel directly without carrying containers. |

Refuel amounts and drain speed per container type are configurable in `mod_config.json`. See [Global Settings](/server-config/global-settings#refuelling) for the full table.

### Refuel Settings

| Setting | What it does | Default |
|---------|-------------|---------|
| `CZH_refuelOnRestart` | `1` = helicopter refuels to full on each server restart. Does not affect fresh spawns — admin-spawned helicopters always respect the spawner's fluid choice. | `1` |
| `CZH_refuelDistance` | Max distance in metres for the refuel action to appear | `4.2` |

### Fuel Warning System

Audio warning beeps alert the crew when fuel is running low:

- Beeps begin at the configured fuel fraction threshold
- The beep interval **shortens as fuel continues to drop** — urgency increases
- Both pilot and co-pilot hear the warning

| Setting | What it does | Default |
|---------|-------------|---------|
| `CZH_fuelWarnThreshold` | Fuel fraction at which beeps begin. `0.20` = starts at 20% remaining. | `0.20` |
| `CZH_warningVolume` | Volume of the fuel warning beep and rotor emergency alarm | `1.0` |

<Note>
  Audio volume settings (`CZH_engineVolume`, `CZH_rotorVolume`, `CZH_warningVolume`, `CZH_crashVolume`) are configured per variant in each helicopter's JSON file. See [Per-Helicopter Settings](/server-config/per-helicopter-settings#audio) for the full table and per-variant tuning details.
</Note>
