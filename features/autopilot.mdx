# Autopilot & Auto-Hover

The autopilot system allows pilots and co-pilots to lock the helicopter into a stable, stationary hover at the press of a button.

## Activating Autopilot

Press **`X`** while in the pilot or co-pilot seat.

**What happens:**
1. The helicopter smoothly decelerates from its current speed
2. It transitions into a locked hover
3. Once stable, it holds **altitude**, **heading**, and **XY position**

The HUD displays autopilot status so both pilot and co-pilot can see whether it is active.

## Disengaging Autopilot

Press **`X`** again, or apply any control input — autopilot releases immediately and returns full control to the pilot.

## Automatic Autopilot Engagement

Autopilot engages automatically if the **pilot voluntarily exits the helicopter while airborne**. This prevents the aircraft from flying away unmanned.

<Warning>
  If the pilot is **killed while in their seat**, autopilot does **not** engage. The helicopter continues on its last heading until someone takes control or it crashes.
</Warning>

## Server Configuration

| Setting | What it does | Default |
|---------|-------------|---------|
| `CZH_autoPilotEnabled` | `1` = autopilot works normally. `0` = disabled server-wide — the `X` key does nothing. | `1` |

Set `CZH_autoPilotEnabled = 0` in `mod_config.json` to disable autopilot on your server entirely.
