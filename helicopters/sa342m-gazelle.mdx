# SA342M Gazelle

The SA342M Gazelle is a sleek 4-seat helicopter with one unique characteristic: its cockpit is reversed compared to all other helicopters in the mod. The longer startup time reflects its more complex simulation.

## Reversed Cockpit

<Warning>
  The Gazelle's cockpit layout is the **opposite** of every other helicopter in this mod.

  - **Pilot seat** → **RIGHT side**
  - **Co-pilot seat** → **LEFT side**

  Keep this in mind when assigning crew or using side-specific features.
</Warning>

## Door Latching

The Gazelle has a door latching system that affects the order in which doors can be opened:

- **Right rear door** requires the **pilot (right front) door** to be removed first
- **Left rear door** requires the **co-pilot (left front) door** to be removed first

This is a physical constraint — you cannot open the rear doors without first removing the corresponding front door.
