The MH-60L Blackhawk is a capable 8-seat medium transport helicopter. Along with the UH-1H, it is one of two aircraft with full Interior Walk. It also features independently toggleable sliding cockpit windows for both pilot and co-pilot, synced across the network.

## Interior Walk

The Blackhawk supports the same Interior Walk system as the UH-1H:

1. Pilot or co-pilot uses **"Move to Cabin"** to enter walk mode
2. Full WASD movement, weapon use, and ADS inside the cabin
3. **"Return to Cockpit"** re-seats them (prefers original seat)

See [Interior Walk](/features/interior-walk) for full details.

## Sliding Windows

The Blackhawk has independently toggleable sliding windows on each cockpit door:

- **Pilot** and **co-pilot** each control their own window independently
- Look at the **cockpit side glass** to get the toggle action
- Window state is **synced across the network** — all players see the same open/closed state
- Can be opened or closed at any time, in flight or on the ground
