## Flight Controls

These controls apply while seated as **pilot or co-pilot**.

| Key | Action |
|-----|--------|
| `W` | Pitch forward (nose down, gain forward speed) |
| `S` | Pitch back (nose up, slow down / reverse) |
| `A` | Bank left (roll left) |
| `D` | Bank right (roll right) |
| `Q` | Yaw left (rotate left on pedals) |
| `E` | Yaw right (rotate right on pedals) |
| `Shift` | Increase collective (climb) |
| `Z` | Decrease collective (descend) |
| `X` | **Auto Hover toggle** — decelerates to a locked hover and holds altitude, heading, and position. Press again or apply input to disengage. Also used for [ground flip recovery](/features/flight-model#ground-flip-recovery) when grounded and inverted. |

## Engine & Lights

| Action | Who | Notes |
|--------|-----|-------|
| Start Engine | Pilot, Co-pilot | Hold action (~1 sec). Requires charged `HelicopterBattery` in battery slot. |
| Stop Engine | Pilot, Co-pilot | Hold action (~1 sec). Blocked if the helicopter is airborne and moving — you cannot cut the engine mid-flight. |
| Toggle Spotlight | Pilot only | Hold action. Long-range white searchlight. Requires battery charge to turn on. |
| Toggle Navigation Lights | Pilot only | Hold action. Port (red), starboard (green), tail (white) and anti-collision beacon (red). Requires battery charge to turn on. |

<Note>
  Both light types drain the battery at `CZH_batteryLightDrain` units per second while on. If the battery dies while lights are on, the lights will turn off automatically.
</Note>

## Crew & Passenger Actions

| Action | Who | Helicopter | Notes |
|--------|-----|------------|-------|
| Move to Cabin | Pilot, Co-pilot | UH-1H, Blackhawk | Enters [Cabin Mode](/features/cabin-mode) — walk freely inside the cabin |
| Return to Cockpit | Cabin Mode player | UH-1H, Blackhawk | Returns to cockpit seat from Cabin Mode |
| Toggle Sliding Window | Pilot, Co-pilot | Blackhawk only | Look at cockpit side glass for the action — instant action, no hold |
| Open / Close Rear Ramp | Pilot, Co-pilot, nearby players | AW101 Merlin only | Must be grounded |
| Raise / Lower Landing Gear | Pilot only | AW101 Merlin only | `F` key — raise after takeoff, lower before landing |
| Remove / Replace Door | Any crew | MD500, SA342M, UH-1H, MH-60L Blackhawk | Doors become physical items |

## Repair Actions

All repair actions require `HelicopterRepairKit` held in hands, engine off, and helicopter grounded.

| Action | Who | Hold Time | Notes |
|--------|-----|-----------|-------|
| Repair Main Rotor | Any player | 45 sec (default) | Player must be standing (not crouched/prone). One repair level per hold. |
| Repair Tail Rotor | Any player | 45 sec (default) | Same conditions as main rotor. |
| Repair Body Zone | Any player | 30 sec (default) | Repairs fuselage damage. One level per hold. |
| Repair Engine Zone | Any player | 30 sec (default) | Only available when main rotor is destroyed. |
| Repair Chassis Zone | Any player | 30 sec (default) | Repairs landing gear/skid damage. |
| Repair Fuel Zone | Any player | 30 sec (default) | Repairs fuel tank damage. |

## Refuelling

| Action | Who | Notes |
|--------|-----|-------|
| Refuel | Any player | Hold a fuel container (e.g. Jerrycan), approach within `CZH_refuelDistance` metres (default 4.2 m), and look at the **fuel refill point** on the helicopter to trigger the action |

The refuel action appears on a specific component on the helicopter's model — the physical fuel filler point. You must be holding a compatible fuel container (gasoline only), within range, and looking at that point for the action to show.

## Cabin Mode Controls

While in Cabin Mode (UH-1H / Blackhawk):

| Input | Action |
|-------|--------|
| `W / A / S / D` | Walk inside the cabin |
| Mouse | Look around |
| All weapon actions | Fully functional — draw, aim (ADS), fire |
| Return to Cockpit action | Re-seats you in your cockpit seat |
