CZ Autoz Helicopter is a complete helicopter simulation built from the ground up for DayZ. Six aircraft, each engineered around its own real-world weight, drag, and rotor characteristics. A damage model with consequences. A configuration system deep enough for any server's vision.

Nothing is shared across helicopters to cut corners — the Merlin feels nothing like the MD500, because it wasn't built to.

<Note>
  Every feature on this page is included in the mod. No additional purchases, no DLC splits, no feature tiers.
</Note>

## Standout Features

### Interior Cabin Walk

Players aboard the UH-1H Iroquois and MH-60L Blackhawk can do something no vanilla DayZ vehicle supports — get up and move around inside a **moving helicopter**.

The pilot or co-pilot triggers a seat exit that keeps them physically attached to the aircraft's coordinate space. From there they can walk the full cabin length with WASD, draw and aim weapons, interact with cargo, and return to the cockpit at any time via a prompt.

<Tip>
  Cargo seats have their own seat-exit walk behaviour — passengers can stand up while airborne and remain linked to the aircraft until landing.
</Tip>

---

### Autorotation

When the engine dies in flight, the rotor keeps spinning — rotor inertia carries the aircraft into a glide. A controlled descent, a collective flare at the right moment, and a heavy helicopter can be put down survivably.

The autorotation model has configurable parameters: gravity reduction factor, flare strength, maximum descent speed cap, and the speed at which the rotor stops providing meaningful lift. Servers can tune it from forgiving to unforgiving.

---

### Grab-Hold System (MH-6 Little Bird)

The MH-6's bench seats work differently from every other helicopter in the pack. When a player sits on a bench, they are physically **linked to the helicopter's local space**. The aircraft can bank, pitch, and accelerate — bench riders move with it automatically.

No manual grab inputs. No risk of falling through geometry. The link is maintained from takeoff to landing and detaches cleanly when the player dismounts.

---

### Damage System

All six zones can be enabled or disabled **per helicopter, per server** in the JSON config. A helicopter with all zones at `0` takes no damage at all — useful for PvE or event servers that want aircraft to be indestructible.

| Zone | Consequence when destroyed |
|------|---------------------------|
| **Body** | Progressive HP drain; cosmetic damage |
| **Engine** | Cannot restart; rotor spins down |
| **Chassis** | Structural damage; impulse threshold lowered |
| **Fuel** | Tank leak; fuel drains continuously |
| **Main Rotor** | Aircraft falls; no lift recovery |
| **Tail Rotor** | Uncontrolled yaw spin; heading cannot be held |

Hard-impact crash damage to crew is also configurable — enabled, disabled, or tuned by impulse threshold and severity scale.

## Server Configuration

Every aircraft in the pack has its own JSON config file generated automatically on first server start. Parameters are applied immediately on respawn — no server restart required after editing.

```
$profile:\CZAutoz_Heli\
├── MD500_Civilian_config.json
├── UH1H_Nasa_config.json
├── MH60LBlackhawk_config.json
├── AW101Merlin_RoyalAirForce_config.json
├── ... (20 files total across all variants)
└── mod_config.json          ← global settings
```

**45+ parameters per helicopter** cover:

- Rotor spool rate, lift multiplier, altitude ceiling
- Cyclic authority (pitch / bank / yaw strength)
- Input smoothing, drag, mass-based inertia
- Ground effect and ETL (effective translational lift) bonuses
- VNE airspeed and vertical speed caps
- All six damage zone toggles plus contact damage coefficient
- Battery drain/charge rates, fuel burn rate
- Audio volume multipliers for engine, rotor, warning, and crash sounds
- Camera distance, speed display unit factor

<Tip>
  The config system is additive — when a mod update adds new parameters, they are automatically appended to your existing files with their default values. Your custom values are never overwritten.
</Tip>

See [Per-Helicopter Settings](/server-config/per-helicopter-settings) for the full parameter reference.

## Logging & Administration

When enabled in `mod_config.json`, the mod writes timestamped activity logs to the server profile directory:

```
$profile:\CZAutoz_Heli\CZAutoz_Heli_Activity_YYYY-MM-DD.log
```

Every log entry includes the **player display name**, **Steam GUID**, helicopter type, and world position. Logged events include engine start, engine stop, boarding, disembarking, and pilot-in-seat death.

A separate debug log (`CZAutoz_Heli_debug.log`) can be enabled independently for detailed physics and config diagnostics without flooding the activity log.

Both logs are opt-in and disabled by default.
